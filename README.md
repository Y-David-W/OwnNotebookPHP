# OwnNotebookPHP
Es un cuaderno personal donde guardo los ejercicios de clase, es muy útil porque cuando creo un archivo o carpeta en el index crea una ruta en el index del proyecto o archivo, también hay un buscador donde puedes buscar archivo específico y previsualizar los archivos
Is a personal nootebook where I save my excercise from class, very useful because when you create a new file or folder in the index can create a navigator route and is essier find the file or project, also we can preview the file and search bar

Nota/Note:
Tienes que tener XAMPP o APACHE instalado/ You must have XAMPP or APACHE instaled.
No tiene todo los archivos que hice ya que son personal, pero sí algunos ejemplos hecho en clase/97 / It doesn't have all the files I made since they are personal, but it does have some examples made in class.


## 1. Descargar el archivo (Fecha)notebook.zip/Download the (Date)notebook.zip
Mueve el archivo xampp o lampp > htdocs o en tu entorno/Move the archive to  xampp or lampp > htdocs or in your development environment
## 2.Ve al local host donde esté tu servidor/go to the local host where is host your server
Puedes cambiar el fav icon o el index por otro en el phpini.conf/You can change the icon or index in the phpini.conf
## 3. Subir una carpeta/upload a folder
Puedes subir carpeta o archivo y después refresca la página/You can upload a folder or document, when is done you should refresh
## 4. Cambiar diseño/change design
Puedes cambiar el diseño como el color o fondo en style.css/You can change the design like color or background in style.css
